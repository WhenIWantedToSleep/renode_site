What is Ren'ode?

This is a program for quickly creating scripts
for visual novels made on the Ren’Py engine.

The work in the program takes place by
creating and connecting nodes,
which are subsequently exported
to the novel project on Ren’Py.

How do I start working with Ren'ode?

To work comfortably with Ren’ode, you need to have the Ren’Py engine installed on your device.
Then install the Ren’ode itself using the links below

The program has a built-in tutorial that
will introduce you to how it works.
